Seletores CSS são padrões usados para selecionar e aplicar estilos a elementos específicos do HTML. Existem diversos tipos de seletores, incluindo:
Seletores de tipo: aplicam estilos a um tipo específico de elemento, como p, h1, div.
Seletores de classe: utilizam a notação de ponto (.) para aplicar estilos a elementos com uma classe específica, como .classe.
Seletores de ID: utilizam a notação de hash (#) para aplicar estilos a um elemento com um ID específico, como #id.
Seletores de atributo: aplicam estilos a elementos que possuem um determinado atributo, como [type="text"].
Seletores pseudo-classe: aplicam estilos com base no estado de um elemento, como :hover, :focus.
Seletores pseudo-elemento: aplicam estilos a partes específicas de um elemento, como ::before, ::after.
